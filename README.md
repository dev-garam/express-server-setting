# node express server default setting

## description
- 개인적으로 설계한 서버 보일러 플레이트 코드입니다.
- domains 를 기준으로 router를 구성하도록 설계하였습니다.
- winston library 를 활용하여 logger 를 관리하도록 구성하였고, swagger 는 yaml 파일로 작성할 수 있도록 설계하였습니다.
